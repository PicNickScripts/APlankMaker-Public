## What It Does

APlankMaker makes planks at supported sawmills with a clean setup flow, Simple auto-routing, and Advanced controls for players who want to choose logs, routes, transport, equipment, and runtime behavior.

## Supported Sawmills

- Auburnvale
- Prifddinas
- Woodcutting Guild
- Varrock Lumber Yard

## Core Features

- Simple mode for owned-supply plank making with automatic route selection
- Advanced mode for manual log, sawmill, transport, gear, and runtime controls
- Supports oak, teak, mahogany, rosewood, ironwood, camphor, and normal logs where available
- Humanized mouse, idle, off-screen, AFK, mini-AFK, and break behavior
- Runtime overlay with quick edit and report controls
- Ironman-safe mode disables Grand Exchange actions
- Target 0 means unlimited

## Experimental Options

- Grand Exchange restocking and selling
- Lunar Plank Make
- Lunar altar switching
- Lunar safe-world hopping

## Recommended Start

For the most stable experience, start with Simple mode or Advanced sawmill mode. Leave experimental options disabled until you are comfortable testing newer workflows.
