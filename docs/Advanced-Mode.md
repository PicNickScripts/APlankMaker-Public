# Advanced Mode

Advanced Mode gives more control over sawmill plank making while keeping the same core sawmill workflow.

## What It Controls

- Log type
- Sawmill route
- Transport preference
- Gear preference
- Plank target
- Break behavior
- Optional experimental restocking and selling

## Recommended Use

Use Advanced Mode when you know which sawmill and log type you want to run.

## Sawmill Selection

Available sawmill choices include Auburnvale, Prifddinas, Woodcutting Guild, and Varrock Lumber Yard. Some routes require account unlocks or items.

## Gear Preference

Auto/Graceful-style weight reduction is the recommended default. Manual gear choices are optional.

## Experimental GE Options

Grand Exchange restocking and selling are optional experimental workflows for normal accounts. Ironman mode disables GE actions.
