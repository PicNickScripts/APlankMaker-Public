# Getting Started

## Before Starting

Have logs and coins available in your bank or inventory. For the smoothest first run, start near a bank and use Simple Mode.

## First Run Setup

1. Start APlankMaker from DreamBot.
2. Pick Simple Mode for automatic owned-supply plank making.
3. Set plank target to `0` for unlimited, or enter a number to stop after that many planks.
4. Keep Ironman mode enabled only if the account should never use Grand Exchange features.
5. Press Start.

## Best Stable Setup

- Mode: Simple or Advanced
- Restock: Off unless intentionally testing it
- Selling: Off unless intentionally testing it
- Lunar Plank Make: Off unless intentionally testing it
- Target: `0` for unlimited

## What Target 0 Means

`0` means unlimited. The script will continue until supplies run out, a stop condition is met, or you stop it.
