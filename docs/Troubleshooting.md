# Troubleshooting

## Script Does Not Start

Check that DreamBot can see the script and that the script was refreshed after updating the jar.

## Wrong Mode Or Old Overlay Appears

Restart the script. If DreamBot had an older local jar loaded, restart the client or refresh Local Scripts.

## Missing Supplies

Confirm logs, coins, and route items are available in bank or inventory. Lunar mode also needs runes and spellbook access.

## Ironman Account Uses GE

Enable Ironman mode. Ironman mode disables Grand Exchange actions.

## Lunar Mode Does Not Cast

Confirm Lunar spellbook, Magic level, Dream Mentor requirement, logs, coins, nature runes, astral runes, and an earth rune source.

## Report A Bug

Use the Report Bug button on the overlay when possible. It creates a local sanitized report you can copy or save, review, and manually share. Include the mode, selected logs, route, and what the script was doing.
