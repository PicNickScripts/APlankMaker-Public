# APlankMaker Guide

APlankMaker is a DreamBot plank-making script by PicNick. It is built around stable sawmill plank making first, with optional experimental workflows for users who want to test newer features.

## Recommended First Run

Use Simple, Advanced, or Lunar Plank Make with the required owned supplies. Leave experimental GE restocking and selling off until you are comfortable testing them.

## Supported Sawmills

- Auburnvale
- Prifddinas
- Woodcutting Guild
- Varrock Lumber Yard

## Supported Logs

- Normal logs
- Oak logs
- Teak logs
- Mahogany logs
- Rosewood logs
- Ironwood logs
- Camphor logs

Some logs or routes depend on account unlocks and current game content availability.
