# Overlay And Bug Reports

## Overlay

The runtime overlay shows current status, plank count, rate estimates, coins, runtime, break status, and mode-specific information.

## Buttons

- Settings: Opens the runtime settings editor.
- Report Bug: Opens a local-only report prompt. It can copy or save a sanitized text report for you to manually share in Discord.
- Discord: Opens the APlankMaker Discord invite.

## Good Bug Reports

Post bug reports in the Discord `#report-a-bug` text channel. Attach the saved report or paste the copied report, then add a brief description that includes:

- Mode used
- Location
- Log type
- Sawmill or Lunar setting
- What happened
- What you expected
- Whether it was a fresh start, bank start, or mid-run issue
- Whether the issue repeated

The report button does not automatically send reports, screenshots, or logs. Use Copy Report or Save Report, review the sanitized text, then manually paste or attach it in Discord. Runtime profile and seed details are redacted from the generated report.
