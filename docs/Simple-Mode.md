# Simple Mode

Simple Mode is the recommended default. It is designed for owned supplies and automatic routing.

## What It Does

- Uses logs and coins you already own.
- Chooses a supported log tier automatically.
- Selects an accessible sawmill route.
- Keeps normal break and idle behavior active.
- Avoids Grand Exchange actions when Ironman mode is enabled.

## When To Use It

Use Simple Mode when you want to start plank making with minimal setup.

## Suggested Settings

- Plank target: `0` for unlimited, or a specific stop amount.
- Gear preference: Auto or Graceful-style weight reduction.
- Restock: Off for the most stable owned-supply run.
- Ironman mode: On for Ironman accounts or any account that should avoid GE actions.

## Notes

Simple Mode prioritizes stability and owned-supply processing. Use Advanced Mode if you want manual control over logs, routes, transport, or runtime behavior.
